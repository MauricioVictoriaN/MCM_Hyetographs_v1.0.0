# MCM_Hyetographs_v1.0.0
Open-source R script generating synthetic sub-hourly design hyetographs from daily precipitation. Combines six-test stationarity ensemble, GEV/GEV-GAMLSS frequency analysis with ONI covariate, Bell (1969) IDF curves, and MCM disaggregation. Exports Q10/Q50/Q90 hyetographs for 9 return periods in HEC-HMS format.
